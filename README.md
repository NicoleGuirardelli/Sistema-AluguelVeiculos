# Sistema de Gestão de Frota de Veículos

Projeto em **Java** simula o gerenciamento de uma frota de veículos de uma locadora.
## Funcionalidades
- Gerencia carros, motos e caminhões
- Controla se o veículo está disponível ou alugado
- Calcula automaticamente o valor do aluguel
- Permite reajuste de preços
- Realiza consultas e filtros:
  - veículos pelo tipo  
  - disponíveis ou alugados  
  - mais barato e mais caro
- Impede ações inválidas, como tentar alugar um veículo já locado
